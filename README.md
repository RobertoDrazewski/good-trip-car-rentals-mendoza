# Mendoza Rent Pro - Premium Car Rental System 🚗🍷

A professional, full-stack car rental management system designed for **Good Trip Car Rentals** in Mendoza, Argentina. This platform features a high-end landing page for customers and a powerful "Super Admin" Dashboard for real-time fleet and booking management.

## 🌟 Key Features

### 💎 Customer Experience (Frontend)
- **Dynamic Booking Engine:** Real-time car availability and pricing calculation.
- **Interactive Routes:** Discover Mendoza's best wineries and landscapes with AI-curated descriptions.
- **Multi-language Support:** Ready for international tourists (English/Spanish).
- **Premium UI:** Smooth animations and mobile-first design.

### ⚡ Super Admin Dashboard
- **Live Fleet Control:** Manage vehicles, status (Available/Busy), and technical specs.
- **Dynamic Pricing:** Update seasonal rates, USD/ARS exchange rates, and insurance fees in one click.
- **Smart Calendar:** Visual booking overview with car-specific color coding.
- **Staff Management:** Secure invitation system via Resend (Email) for new collaborators.
- **AI Integration:** OpenAI-powered route description enhancer.

## 🛠️ Tech Stack

- **Frontend:** React.js, Tailwind CSS, Lucide Icons, Vite.
- **Backend:** Node.js, Express.js.
- **Database:** MySQL.
- **Integrations:** Resend (Email), Google Maps API, OpenAI API.
